Transcripción Mejores Prácticas del Trabajo Remoto

¡Saludos!

Les doy la bienvenida al tema “Mejores Prácticas del Trabajo Remoto”.

Algunas buenas prácticas al trabajar de manera remota se citan a continuación:

  • Red privada virtual o VPN: El VPN permite extender una Red de Área Local (LAN) de manera segura sobre una red pública o no controlada como Internet. Se recomienda utilizarlo para evitar que un ciberdelincuente espíe las comunicaciones realizadas.

  • Utilizar el escritorio remoto a través de VPN: Con esta modalidad de VPN se logra que una red de oficina esté disponible de forma remota para los usuarios autorizados. Para evitar riesgos de vulnerabilidades o configuraciones inadecuadas, se recomienda usar el escritorio remoto solo a través de esta modalidad.

  • Utilizar, de preferencia, dispositivos corporativos: Cuentan con las políticas de seguridad y el software establecidos por la misma empresa.

  • En caso de utilizar dispositivos personales debe ser bajo la política BYOD.

  • Entorno de trabajo seguro: Respetar las políticas de protección del puesto de trabajo en cualquier lugar.

  • Conexión a internet protegida: Evitar redes Wi-Fi públicas.

  • Datos móviles como plan b: Cuando no se puede utilizar una red segura, recurrir al uso de datos móviles 4G o 5G.

  • Periodo de implantación y pruebas: Valorar distintos escenarios y configurar el dispositivo antes de comenzar con el teletrabajo.

  • Elaborar una política de teletrabajo: Esta medida incluye la formación de quienes trabajan en la empresa para que puedan seguirla de manera adecuada.
